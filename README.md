📌 Project Overview
Performed Exploratory Data Analysis (EDA) on Diwali Sales dataset
Analyzed customer behavior, sales trends, and purchasing patterns
Objective: Identify key factors influencing sales during festive season

🎯 Objectives
Understand customer demographics (age, gender, marital status)
Identify top-performing states and product categories
Analyze purchasing power and buying patterns

🛠️ Tech Stack
Python
Pandas (Data Cleaning & Analysis)
NumPy
Matplotlib & Seaborn (Data Visualization)

📂 Dataset Information
Total Records: 11,251
Features: Customer details, product category, orders, amount, state, occupation
Data cleaning performed (null values removed, columns dropped, datatype fixed)

🔍 Data Cleaning Steps
Removed unnecessary columns (Status, unnamed1)
Handled missing values using dropna()
Converted Amount column to integer
Renamed columns for better understanding

📊 Key Analysis Performed
Gender-wise sales analysis
Age group vs purchasing behavior
State-wise sales and orders
Marital status impact on spending
Occupation-based analysis
Product category performance

📈 Key Insights
Female customers have higher purchasing power than males
Age group 26–35 contributes the most to sales
Top states: Uttar Pradesh, Maharashtra, Karnataka
Married women are the highest buyers
Major occupations: IT, Healthcare, Aviation
Top categories: Food, Clothing, Electronics

🧠 Conclusion
Target audience: Married women (26–35 age group)
High sales regions: UP, Maharashtra, Karnataka
Focus categories: Food, Clothing, Electronics
Helps businesses improve marketing and sales strategies
